호스팅

호스팅(Hosting)이란 서버 컴퓨터의 전체 또는 일정 공간을 이용할 수 있도록
임대해 주는 서비스를 말한다. 사용자가 직접 서버를 구입하고 운영할 필요 없이
호스팅 업체가 미리 준비해 놓은 서버를 빌려 사용하는 형식이다. 호스팅 서비스에는
웹 호스팅과 서버 호스팅, 메일 호스팅 등 다양한 종류가 있다.

    가상 서버 호스팅

    가상 서버 호스팅은 서버의 일정한 공간을 KVM, Xen, OpenVZ(패러렐즈의 Virtuozzo를 오픈소스화한 것) 등의 가상화 기술을 이용하여 한대의 서버처럼 나눠 주는 서비스이다.
    얼핏 보면 아무 문제없어 보이지만, 오버셀링(Overselling)으로 총칭되는 서버 자원을 한계 이상으로 파는 행위에 악용되기도 한다. KVM, Xen도 오버셀링이 없는 건 아니지만,
    특히 OpenVZ는 오버셀링에 주로 연루되는 가상화 솔루션이다. 이미 해외에서는 이 가상화 기술로 오버셀링하는 업체가 우후죽순으로 생겨나서 이미지가 나락으로 떨어진 지 오래다.
    가상 서버를 주문할 때 유의할 필요가 있다. 가상 서버 호스팅은 가상 서버(Virtual Private Server) 하나를 고객에게 통째로 제공해 주는 호스팅 방식이다. KVM, OpenVZ, Xen 등의 가상화 기술을 활용하여 한 대의 서버처럼 쓸 수 있게 해 준다.
    웹 호스팅보다는 제약이 적지만 리소스를 공유하는 것이기 때문에, 중요한 서비스에는 가상 전용 서버 호스팅, 전용 서버 호스팅, 또는 코로케이션을 사용하는 것이 좋다. 클라우드 컴퓨팅이 본 궤도에 오르면서 사용률이 높아졌다. 클라우드 업체들에서는 VPS, VDS, 가상 서버라는 용어 대신에 컴퓨트 인스턴스라는 명칭을 사용한다.
    다만, 자원을 공유하는지(VPS) 혹은 독점적으로 사용하는지(VDS)에 대해 명확히 표시하지 않는 경우가 많으므로 유의할 필요가 있다.

    가상 전용 서버 호스팅

    가상 전용 서버 호스팅(Virtual Dedicated Server, VDS)과 가상 서버 호스팅(VPS)의 차이점을 간단히 설명하면, 가상 서버 호스팅(VPS)은 독점 코어(CPU)가
    아닌 공유 코어(CPU)를 제공하나, 가상 전용 서버 호스팅(VDS)은 독점 코어(CPU)를 제공하므로 CPU 사용률 제약에서 자유롭다는 점이다. 주의해야 할 점은, 전용
    서버 호스팅과는 엄연히 다르고, 일부 업체는 이 호스팅을 전용 서버 호스팅으로 둔갑시키기도 한다. VDS를 VPS with Dedicated Cores로 퉁치는 경우도 흔하다.
    한국에서는 독점 코어(CPU)를 주는 가상 전용 서버 호스팅도 별다른 구별 없이 가상 서버 호스팅으로 호칭하는 경우가 많다. 서비스 사양에 CPU가 독점인지 공유인지

    가상 서버 호스팅

    전용 서버 호스팅은 서버 한대를 전부 임대해 주는 서비스이다. 일반적으로 전용 서버가 가상 서버에 비해 비교적 비싸지만, 가상 서버는 여러모로 제약이 많은 편이다.
    얼마나 제약이 있느냐는 호스팅 업체의 정책에 달려 있다. 가상 서버와 전용 서버의 중간격인 가상 전용 서버(Virtual Dedicated Server)라는 것도 있다. 자원을 독점적으로
    사용하는 가상 서버로 제약이 거의 없다. 단독 서버 호스팅이라고도 불리는 전용 서버 호스팅 (Dedicated Server Hosting)은 한대의 서버를 통째로 임대해 주는 서비스다.
    아마존 웹 서비스를 비롯한 다수의 업체들은 물리적으로 독립된 전용 서버 호스팅의 특징을 강조하기 위해서 인스턴스에 Metal 혹은 Bare Metal(베어 메탈)이라는 수식어를 붙이기도 한다.
    기본적으로 가격이 월 8만원대부터 시작하기 때문에 다른 호스팅보다 비싸지만, 제약이 거의 없고 서버 자원을 혼자서 모두 독점적으로 사용할 수 있다는 장점이 있다.
    표기해 놓으므로 구별은 가능하다.
